# Function Basics

Create a function that gets 2 numbers as parameters and returns the
greater one. The function should not break with negative numbers.

## Expected results:
| Input  | Output |
|--------| -------|
| 1, 5   |      5 |
| 5, 1   |      5 |
| 1, 1   |      1 |
| -5, 4  |      4 |
| -1, -3 |      4 |
